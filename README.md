# telecom-churn-analysis-issue-tree
Diagnostic churn analysis on IBM's public Telco dataset (7,043 customers). McKinsey-style issue tree across 6 branches, isolates a 989-customer segment churning at 65.8%, and trains a Gradient Boosting model (ROC-AUC 0.842) to rank drivers. Ends in 4 testable A/B hypotheses, not unverified claims of impact.
